# Kerdoiv.ahk
Ez a program végigkattintja helyetted a kérdőíveket
# Használat
Futtatás után a `H` betű lenyomása után a szkript 20-szor fut le:
- kattintás
- lefele nyíl megnyomása

Kérdőív kitöltés után érdemes a tálcáról bezárni a programot
